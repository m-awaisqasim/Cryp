# Frontend
http://localhost:7070

# Run without Sleeping mode:
JARVIS_HOTWORD=0 python3 main.py

# Enable auto-start (runs on every login)
systemctl --user enable jarvis
systemctl --user start jarvis   # start now

# Check status
systemctl --user status jarvis

# Stop / disable
systemctl --user stop jarvis
systemctl --user disable jarvis

# View logs
journalctl --user -u jarvis -f

# Note:
One thing to note: since hotword is enabled by default, it'll boot into sleep mode waiting 
for "Hey Jarvis" — no window pops up until you say the wake word. 
The actual PyQt6 GUI window only appears after the wake word triggers a Gemini session connection.
